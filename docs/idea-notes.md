# Idea Notes

## GitHub/Coder Patterns That Shaped This

- Coder templates are Terraform recipes for repeatable workspaces.
- Coder registry modules make dev tools, IDEs, and apps available inside a
  workspace without hand-installing everything.
- Good Coder demos are not just apps. They show that a workspace can carry the
  app, data collectors, scripts, secrets, and editor setup together.

## Product Idea

Build a personal "should I go?" planner for the Austin/San Antonio corridor.

Most local apps are event feeds. This should be more opinionated:

- What can I combine into one trip?
- Is this worth the drive from Schertz?
- Can I pair live music with food nearby?
- Is there a weird-history stop on the way?
- Is there a garden task I should do before leaving?
- Can code/automation remove the repeated planning work?

## Why It Fits

This combines practical local life with the kind of developer-story project
Coder is good at:

- local seed data now
- structured imports later
- workspace-hosted tools and scheduled jobs
- a personal but broadly understandable demo
- easy story for a meetup: "I used a governed dev workspace to build a local
  lifehacking app before dinner."
